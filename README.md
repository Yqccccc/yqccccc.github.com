## 项目描述</br>
实现了一个各地区防疫政策查询系统移动端，提供全国各地城市中高风险地区个数、封控地区个数以及相关防疫政策内容，同时可显示该地区风险程度以及近期疫情情况。该系统具有查询、切换、跳转等功能。</br>
